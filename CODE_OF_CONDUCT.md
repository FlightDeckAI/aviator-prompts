# Code of Conduct

We aim to keep this community respectful, professional, and helpful.

## Be respectful
Assume good intent. Give constructive feedback. Keep discussions focused on the work.

## No harassment
Harassment, insults, hate speech, sexualized content, threats, or doxxing are not tolerated.

## Enforcement
Maintainers may remove content or restrict participation if behavior violates this policy.

## Reporting
If you experience or witness behavior that violates this policy, open a GitHub issue (if safe) or contact the maintainer via the repository owner profile.
