# Prompt Pack

Copy any prompt template below into your model of choice and fill in the bracketed fields.

## Conventions
- Replace `[BRACKETS]` with your details.
- If you paste raw products (METAR/TAF/NOTAM) keep them as-is.
- If the model must assume anything, require it to **state assumptions**.
